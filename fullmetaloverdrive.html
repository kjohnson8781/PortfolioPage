Full Metal Overdrive

At Deadlift Studios, I'm working as a Technical Artist specializing in Tools and Pipeline on Full Metal Overdrive, a 3rd-person multiplayer combat racing game.

Primary Responsibilities:

  • Develop custom tools by extending functionality of Unreal Engine 5, Houdini, and Maya to address artist challenges, such as tedious manual tasks and a lack of dedicated resources
  • Set up version control integration to automate check in and check out process between Perforce and Unreal Engine 5 and manage project organization for over 4,000 assets
  • Automate asset management, build and release processes, and project validation to increase development iteration speeds and optimize content creation pipeline
  • Identify inefficiencies in artist workflows and collaborate with technical leads to implement solutions that meet user requirements and align with production needs
  • Support 20 game artists and developers by leading training sessions and creating comprehensive tools documentation to ensure team alignment and effective knowledge transfer

Notable Achievements:

  • Developed tool that validated asset naming conventions and identified unused assets, improving project organization and resulting in a 25% reduction in project size
  • Set up a dedicated build machine, wrote scripts to automate nightly test builds, and designed a release distribution pipeline for artists and developers using Unreal Game Sync
