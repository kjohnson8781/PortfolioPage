DailyParking

Ever had to deal with annoying towing companies? Ever forget to register your car with Register2Park? Erase your towing woes with a single script! DailyParking is a Python script that automatically submits your car information for guest parking registration with a single click!

The Python script utilizes geckodriver and selenium to analyze the website HTML code and automatically fill in the data in the correct form fields. In order to make the script run daily, I utilized crontab in Linux. It currently runs on FireFox and is compatible for the Register2Park service.

Personal Home Lab

This one is still in progress... and hopefully always will be! I turned my old laptop into a basic server. I also had the opportunity to obtain SuperMicro server equipment and am working to integrate it into my network. I installed Linux server in order to get more comfortable with Linux CLI.
