Featured Project

Material ID Script for Maya

The Material ID Tool allows a user to apply autogenerated materials from predefined IDs to objects in Maya for visualization and texture assignment. It is primarily used for identification of object materials and parts when moving assets to other DCCs.

Highlights include:
  • - Actively used by artists in the Kyber Initiative project to eliminate manual processes and integrate models to UE5
  • - Reads a text file from any directory on the user computer with the file path editable at any time
  • - Edit the text file directly from the script using Add and Delete buttons
  • - Customizable UI with a scrollable, resizeable window and a custom column size setting that is saved per user preference
  • - Continuously adapted to user feedback by aligning and enhancing features based on artist requirements
