Material ID Script

Python
PyQt5/Pyside2
MEL
Maya

The Material ID Tool allows a user to apply generated IDs to objects in Maya for visualization purposes and easy texture assignment when moving assets to other DCCs.

Features include:

  • - Automatically generate materials and shading groups with random colors for predefined IDs
  • - Apply materials to objects with a single button click
  • - Reset object materials to default 'lambert1' if mistakes are made
  • - Select all objects with a specific material by right clicking the button

The tool was originally created for asset migration to Unreal Engine 5 but is not restricted.

Version Control Tool

Python
PyQt5/Pyside2
VEX
Houdini

The Houdini Version Control Tool allows a user to increment and save a Houdini file with a version suffix automatically.

Features include:

  • - Auto-increment and save versions with _v### suffix (also works with _V### if that is existing format)
  • - Add default _v001 if file does not have version
  • - User has option to overwrite or save as latest version if incremented version already exists
  • - Easily view a list of all versions of the current file via the shelf tool
  • - Seamlessly switch between versions by double-clicking to open desired version from the list

Procedural Asset Generator: Mushrooms

VEX
Houdini

The Procedural Mushroom Generator allows users to procedurally model and colorize a mushroom in Houdini.

Features include:

  • - Create a mushroom model from any user-created curve
  • - Edit stem and cap properties including color, shape, size, and more
  • - Add and edit cap accessories
  • - Add and edit cap noise for more realistic, non-symmetrical shapes
  • - Change curve at any time and all previous changes will remain

Rubik's Cube Script

Python
PyQt5/Pyside2
MEL
Maya

The Rubik Cube python script allows a user to model and manipulate a Rubik's cube in any Maya scene.

Features include:

  • - Create a fully-textured Rubik Cube
  • - Manually rotate any face
  • - Randomize the cube
  • - Solve the cube

All rotations are recorded as an animation which can be watched as a playback video or rendered.

Cloud Resume Challenge

Azure
JavaScript
HTML/CSS

The Cloud Resume Challenge is a project put forth by Forrest Brazeal and was my first attempt at building a personal website.

The website is hosted in Azure and uses Cloudflare for network security. I wrote a JavaScript file that counts the number of visitors to my site and stores it in a database. My data is stored CosmosDB and connected via a serverless API using Azure functions.

I've since revamped my website but I learned a ton about cloud and web technologies in this project.
